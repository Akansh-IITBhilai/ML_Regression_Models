# ML Regression Models Summary

## Project Overview

This Jupyter notebook implements a comprehensive machine learning project for predicting median house prices in California using the scikit-learn California housing dataset. The project follows a professional workflow covering data preparation, preprocessing, and evaluation of multiple regression models.

## Models Evaluated

- OLS Linear Regression
- Ridge Regression (L2 regularization)
- Lasso Regression (L1 regularization)
- Decision Tree Regressor

## Workflow Steps

1. Data Collection
2. Initial Exploratory Data Analysis (EDA)
3. Data Cleaning (duplicates, data types, missing values)
4. Numerical Analysis and Outlier Detection
5. Outlier Treatment (Winsorization)
6. Data Standardization
7. Multicollinearity and VIF Analysis
8. Feature Selection
9. Train-Test Split
10-13. Model Training (OLS, Ridge, Lasso, Decision Tree)
14. Model Evaluation and Comparison
15. Graphical Interpretation and Reporting

## Key Findings

The Tuned Ridge Regression model showed the best performance with optimal balance of error metrics and explanatory power.
